## ⚠️ SEO BEST PRACTICES & AVOID LIST

### DO:
✅ Write for humans first, search engines second
✅ Use keywords naturally in content
✅ Create unique, valuable content
✅ Optimize for user experience
✅ Build quality, relevant backlinks
✅ Update content regularly
✅ Respond to all reviews
✅ Use mobile-friendly design
✅ Test structured data markup
✅ Monitor performance metrics

### DON'T:
❌ Keyword stuffing (overusing keywords)
❌ Duplicate content across pages
❌ Buy backlinks or participate in link schemes
❌ Hidden text or links
❌ Cloaking (showing different content to users vs search engines)
❌ Make false medical claims
❌ Ignore mobile optimization
❌ Forget to add alt text to images
❌ Use irrelevant keywords
❌ Neglect page speed optimization

## 4. ON-PAGE SEO BEST PRACTICES

### Headings Structure:
```
H1: Main topic (one per page)
H2: Main sections
H3: Subsections
H4: Details
```

### Content Optimization:
- Use keyword naturally (1-2% keyword density)
- Write for humans first, search engines second
- Use internal linking to related pages
- Include long-tail keywords in content
- Answer user questions in content
- Use bold and italic for emphasis
- Break content into scannable sections

### Image Optimization:
- Use descriptive file names: `kidney-stone-treatment-lahore.jpg`
- Add descriptive alt text: `"Dr. Muneeba Malik Awan treating kidney stones using homeopathy"`
- Compress images for faster loading
- Use WebP format when possible

## 9. KEYWORD OPTIMIZATION TIPS

### For Title Tags:
- Include main keyword within first 60 characters
- Add location modifier (Lahore)
- Include brand name when possible
- Format: "[Service] - [Doctor Name] | [Location]"

### For Meta Descriptions:
- Natural inclusion of 1-2 keywords
- Include call-to-action
- Keep under 160 characters
- Make it compelling and unique per page

### For Content:
- Use keyword in first 100 words
- Place in H1, H2 tags naturally
- Include variations and synonyms
- Aim for 1-2% keyword density
- Answer user search intent

### For URL Structure:
- Use hyphens: `/kidney-stone-treatment-lahore.html`
- Keep URLs short and descriptive
- Include main keyword when possible
